단조 스택
1. 📚 단조 스택(Monotonic Stack)이란?
   **"스택 내부의 원소들이 항상 '오름차순' 혹은 '내림차순' 정렬 상태를 유지하도록 강제하는 스택"**입니다.

Monotonic (단조): 수학 용어로 "방향이 바뀌지 않고 한쪽으로만 간다"는 뜻입니다. (계속 증가하거나, 계속 감소하거나)

핵심 규칙: 새로운 값이 들어오려고 할 때, **정렬 규칙을 깨뜨리는 기존 놈들을 전부 제거(Pop)**하고 들어갑니다.

🛠️ 탑 문제에서의 적용 (내림차순 단조 스택)
탑 문제는 왼쪽에서 **"나보다 높은 첫 번째 탑"**을 찾는 것입니다. 따라서 스택은 **[9, 7, 4]**처럼 **내림차순(점점 작아지는 형태)**을 유지해야 합니다.

왜? 만약 스택에 [9, 3]이 있는데 7이 들어온다고 칩시다.

3은 7보다 작아서 7에게 가려집니다. 미래에 어떤 탑이 와도 3을 볼 일은 없습니다.

그러니 3은 과감히 **삭제(Pop)**하고 7을 넣어서 [9, 7]로 만드는 게 이득입니다.

2. ⚡ 알고리즘 동작 3단계 공식
   이 공식은 그냥 외우셔도 좋습니다. 모든 단조 스택 문제에 똑같이 쓰입니다.

숙청 (While Pop): 나(현재 값)보다 **약한 녀석(조건에 안 맞는 값)**들은 스택에서 다 내쫓습니다.

기록 (If Peek): 다 쫓아내고 나니 내 바로 앞에 **강한 녀석(Stack Top)**이 남아있다? 걔가 바로 **정답(수신탑)**입니다.

남은 게 없다면? 내가 짱이니 수신탑은 0입니다.

입장 (Push): 이제 나도 스택에 들어가서 다음 신입을 기다립니다.


단조 스택 -> 숙청,기록,입장
